## Getting Started

Requires Node 16+ and Yarn. Newest LTS version of Node is recommended (18+).

1. Install dependencies

```bash
yarn
```

2. Create own `.env.local` file, and replace with your own values

```bash
cp .env.example .env.local
```

3. Start dev server

```bash
yarn dev
```
